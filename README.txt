На странице добавления объявления есть "параметры", которые появляются при выборе "категории"
объявления. Для тог что бы функционал работал нужно:
	в свойствах инфоблока задать поле (типа список) название поля: ADDITION_97_OPT1, где
	97 - ID раздела(категории) объявления,
	OPT1 - порядковый номер HTML элемента select.
	Пример:
	для категории объявлений "грузовые" нужно добавить 2 параметра - пробег, грузоподъемность
	в свойствах инфоблока записываем так:  
		название поля - автомобили грузовые
		тип поля - список
		код свойства - ADDITION_97_OPT1
			в свойствах списка записываем заголовок и перечисляем возможные варианты:
				XML_ID - title  НАЗВАНИЕ - пробег
				XML_ID - 1      НАЗВАНИЕ - 1000км
				XML_ID - 2      НАЗВАНИЕ - 2000км
		
		название поля - автомобили грузовые
		тип поля - список
		код свойства - ADDITION_97_OPT2
			в свойствах списка записываем заголовок и перечисляем возможные варианты:
				XML_ID - title  НАЗВАНИЕ - грузоподъемность
				XML_ID - 1      НАЗВАНИЕ - 10 тонн
				XML_ID - 2      НАЗВАНИЕ - 40 тонн
